We have our own Rust server.

Build your own City or play PVP!

How to Join?
Click HERE to join!

WAY2:
Press F1 when you are ingame and type in: connect discordler.smartlife.tk:28015

More Informations:
- Integrated mini map in the upper left corner with all users on the server an the direction they are heading
- When was RUST last time online? Click HERE to check the Status

Commands
/kit starter | If you died you can start with this starter kit
/map | Show a minimap
/home set "homename" | Set your home base to teleport to it
/friend add "username" | Add friend in Rust
