General Stuff




How to start a game server?
Go to OGP and user your login credentials or use the user you've got and it's password.
Then go to "Game Monitor" and press the game server you want to start. Scroll down until you see "Server starten" / "Start Server". It will now start.

Make superadmin in ULX (Gmod) via OGP
ulx adduser YourName superadmin

Our Windows VM




How to connect to our Windows VM?
We are hosting several Game Server on our Windows Server 2016 Standard VM, because Linux is not supported for some games.
Connect to our Windows VM by starting a Remote Desktop Connection on your pc.
Now enter the following IP Adress and Username "Administrator": discordler.ddns.net


MW3 Ingame Commands




!putgroup[playername],[role] - give a player a role (for ex. mod, adm)
!help - show available commands
!gamemode ,gamemode - Change Gamemode and Gametype
!map - change current map
!map_rotation - lädt die nächste map + gamemode aus "default.dsp"
!res - restart the map quickly to reset teams
!kick [playername] - kick a player (only need to enter part of their name)
!ban [playername] - ban a player (only need to enter part of their name)
