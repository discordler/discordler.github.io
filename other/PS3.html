PS3 Controller PC Einrichtung

Alle Infos findest du hier!
Discord Hilfe


SCP Server
1. Schließe deinen Controller via USB an. 2. Lade dir SCP Server von hier herunter! (WICHTIG: Es wurde beabsichtigt die alte Version genutzt!)
3. Starte SCP Server und Wähle dein "Force Install" und sowohl dein Bluetooth Gerät, als auch dein DS3 Controller
4. Starte Scp Toolit Monitor.exe . Er sollte nun hier erscheinen.
Wenn du links unter Host Adress etwas stehen hast, funktionieren auch deine Bluetooth Treiber bereits, wenn nicht gehe zu "XInput Wrapper - Custom Bluetooth Driver" Wenn dein Controller sich automatisch per Bluetooth Verbinden kann, bist du fertig. Wenn nicht gehe zu "Bluetooth Verbindung"

XInput Wrapper - Custom Bluetooth Driver
NUR WENN NÖTIG: XINPUT WRAPPER IST VERALTET & sollte nur für die Bluetooth Einrichtung genutzt werden, anschließend wieder löschen! 5. Lade dir XInput Wrapper von hier herunter!
6. Wenn du links unter Host Adress etwas stehen hast, funktionieren auch deine Bluetooth Treiber bereits, gehe zu Schritt 7
6.1 Starte ScpServer\bin\Zadig\zadig.exe -> Device -> Load Preset Device -> Bluetooth.cfg
6.2 Options -> List All Devices -> Nun kannst du dein Bluetooth Gerät auswählen (meist etwas mit Bluetooth im Namen)
6.3 Klicke nun auf Replace Driver, warte auf die Fertigstellung und starte deinen PC neu
6.4 Wenn du links unter Host Adress etwas stehen hast, funktionieren deine Bluetooth Treiber

Bluetooth Verbindung
7.1 Stecke nun deinen Controller von USB ab, es sollte kurz "Reserved" da stehen
7.2 Drücke nun ca. 5 Sek. lang den PS Button (die LEDs sollten ausgehen, anschließend wieder an gehen) und der Controller sollte via USB verbunden sein.
-> Stelle sicher, dass du alle Schritte von oben aufmerksam befolgt hast!
-> Stecke deinen Controller ab, wenn Reserved da steht, drücke für eine längere Zeit den PS Button und warte bis zu 1 min.
