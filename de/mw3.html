Allgemeine Infos

Läuft der Server gerade?
Wie Installiere ich Tekno MW3 ohne die Steam Version zu besitzen?
1. Öffne Steam -> Add-Ons (unter Bibliothek, nicht Spiele) -> "Cod MW3 Dedicated Server" herunterladen
2. Steam Ordner Öffnen und den kompletten MW3 Ordner in ein anderes Verzeichnis kopieren/verschieben
3. Klicke oben auf dieser Website auf MW3 Tekno Dateien Download.
4. Kopiere alle Dateien aus dem .zip File in dein MW3 Verzeichnis und bei Nachfrage wähle "Dateien ersetzen"!
5. Fahre bei "Wie trete ich dem Server bei?" fort.

Wie trete ich dem Server bei?
1. Starte die TeknoMW3.exe
2. Klicke auf MP (Direct Connect)
3. Öffne die Einstellungen -> Dedizierte Server -> Beides auf "Ja" setzen
4. Gehe auf Server Liste und suche nach unserem Server ziemlich weit unten
"|Discordler| MW3 DLC+GER"

Ich sehe den Server nicht?
Bitte stelle sicher, dass du die neueste Version installiert hast: 2.8.0.4 ! Mit einer älteren Version findest du unseren Server nicht
Wie behebe ich den Klasseneditor Bug?
Beschreibung: Du kannst keine Waffen, Equipement etc in deinem Klasseneditor auswählen bzw. verändern.
1. Lade dir die Englischen Sprachdateien von HIER herunterladen
2. Kopiere den Ordner english in deinen MW3 Ordner nach /zone
3. Öffne die Datei "localazations.txt" in deinem MW3 Tekno Ordner
4. Änder german zu english

Wie Installiere ich die DLCs
1. Lade dir die Dateien von HIER runter!
UND diese Dateien von HIER runter!
2. Kopiere den Inhalt von dem ersten Download mit den 40 DLCs nach /MW3/zone/dlc
Den rest nach /MW3/main
3. Fertig
